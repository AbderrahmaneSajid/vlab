
      
  










1. Clone the repo
   ```sh
   git clone https://github.com/AbderrahmaneSajid/vlab.git
   ```
2. Install the modules using npm command:
* npm
  ```sh
  cd client
  npm install
  ```
  ```sh
  cd backend
  npm install
  ```
3. Run the local server in `Terminal`
   ```cmd
   cd client
   npm start
   ```
   ```cmd
   cd backend
   nodemon backend
   ```
